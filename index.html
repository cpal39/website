Projects

My projects include two discord bots, a website redesign, a web application, and this very website! The two discord bots are BarryBot and Gif Charades. BarryBot, named after my pet fish, is a basic chatbot that sends different responses to users based on what they say. He also has basic commands such as flipping a coin, rolling a dice, and kicking users. He also a chance of kicking half the members of a server with the command "Barry snap" in honor of Avengers Infinity War. Gif Charades is a simple game where a user types "/gc play" and is then presented with a subject such as "movie" a word count, and a series of gifs. The user then must tell the bot what the title of the subject is. While BarryBot is more for personal use, Gif Charades is on the official Discord bot store.

The website redesign was done during my time as a web development intern for Konica Minolta. Their Supplier Solutions website had not received a major update in 10+ years, and I was tasked with making it look more modern. The only issue was because of how the website was originally set up, I was not able to access any of the html or the css files. I was confined to a single style tag where I had to write css for the website by doing inspect element repeatedly to see the tags and style them by id or attribute.

The web application I worked on was a group final project for a graduate web programming course. Our application was a Marvel/DCEU character database where I was responsible for HTML, CSS, and client-side & server-side JavaScript for the user database system, search results, character details, security, and admin user features. Tools used included handlebars and MongoDB. After using Wix for a couple years to create my website, I decided to create one my own using the skills I have learned from Stevens.

This website is made using a custom version of bootstrap with sass files being compiled by gulp to add my own personality to the syle of the site.
